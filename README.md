## 📝 요구사항 및 분석
1. 회원관리, 상품관리, 결제관리, 주문내역 확인 기능
2. CRUD를 위한 REST API
    - 입력, 수정 시 데이터 형식의 유효성 검사
    - REST API에서 벗어난 param 또는 요청에 대한 응답 처리
    - 상품, 결재, 주문은 관리자만 입력, 수정, 삭제 가능
    - 유저는 회원가입, 결제, 주문 외에는 조회만 가능
    - 조회 결과는 JSON 형태로 응답
